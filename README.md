<h4 align="center">
Prevents Minecraft from pausing when you alt-tab or lose window focus.
</h4>

<h1 align="center">ğŸš€ Getting Started</h1>

### Installation
1. Install **Fabric Loader**
2. Install **Fabric API**
3. Drop the mod into your `mods` folder
4. Launch Minecraft

No setup required. The mod is active immediately after installation.

<h1 align="center">ğŸ“‹ Feature List</h1>

<li>Disables pause on alt-tab</li>
<li>Disables pause on window focus loss</li>
<li>Disables background auto-pause behavior</li>
<li>No config, no toggle, no keybinds</li>
<li>Always enabled while installed</li>
